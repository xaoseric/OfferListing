Welcome to Offer Listings

Offer listings is an open source provider listing system. It uses the Django framework and any SQL database.

Testing

The whole site is covered with Unit Tests. To test the site, run the following commands:

$ fab test
$ fab coverage

The first tests the application, and the second will create the test reports.
